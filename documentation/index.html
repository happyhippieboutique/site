Installing Template

Installing The Template

After you unzip the downloaded pack you'll find a Template Folder with all necessary files.
You can review this Template in any browser, you can display or edit the Template without Internet connection.
Now open your FTP Client (like Filezilla) or directly through channel, upload the content of the Template on your server root.
Once upload of the files is complete go to www.yourdomainname.com/index.html

HTML Srtucture

HTML Structure

This template has responsive layout and is based on Bootstrap Framework. For more information please visit Bootstrap.

The default Bootstrap grid system utilizes 12 columns making 1170px wide container without responsive features enabled. With responsive CSS file added the grid adapts to 724px and 1170px wide container depending on your viewport. The columns become fluid and stack vertically when viewports are below 767px viewports.

 <div class="container">
	<div class="col-md-4">...</div>
	<div class="col-md-4">...</div>
        <div class="col-md-4">...</div>
 </div>

Nesting Columns

To input the content with grid as default, add new .container and a set of .col-md-* columns inside the existing .col-md-* column. Inserted rows should include a set of columns, which would add to the number of rows its parent.

<div class="container">
   <div class="col-md-9">
       Level 1 column
       <div class="row">
           <div class="col-md-6">Level 2</div>
           <div class="col-md-6">Level 2</div>
       </div>
   </div>
   <div class="col-md-3">
       Level 2 column
   </div>
</div>

Menu types

There are 4 different types of menu included

1. Menu transparent

It has been used on Home First Page. For this effect header tag must have next classes:

<header class="header transparent">

This type of menu is not modified to include sub-menu.

2. Menu regular

It has been used on Home Second, Home Five and on biggest part of the pages. For this menu type header tag must have next classes:

<header class="header no-shadow">
3. Menu vertical

It has been used on Home Third Page. For this menu type header tag must have next classes:

<header class="header vertical">
4. Menu half-displayed

It has been used on Home Fourth Page. For this menu type header tag must have next classes:

<header class="header half">

There is a possibility to change menu options as per your demands.

Galery items have the following structure:

Replace url(../path/to/your-file.jpg) with actual path to your image.

Replace Lorem ipsum dolor sit amet with your text.

Gallery filter looks like:

<div id="filters" class="button-group center">
    <button class="btnGallery is-checked" data-filter="*">show all</button>
    <button class="btnGallery" data-filter=".barber">barber</button>
    <button class="btnGallery" data-filter=".nail">nail</button>
    <button class="btnGallery" data-filter=".spa">spa</button>
    <button class="btnGallery" data-filter=".beauty">beauty</button>
    <button class="btnGallery" data-filter=".hair">hair</button>
</div>

To change category name replace text in button tag, for example barber to your category name.

To create your own name category filter, simply in data-filter=".barber" replace barber to your own value.
And add same name class into gallery element-item: 

<div class="gallery element-item your-filter-category-name">

Map

Map block has the following structure:

<!--map-->
<div class="map" id="map-canvas" data-lat="40.677763" data-lng="-73.943869" data-zoom="13" data-styles="2"></div>
<div class="addresses-block">
    <a class="marker" data-lat="40.675653" data-lng="-73.936754"
       data-string="1. Here is some address or email or phone or something else..."></a>
</div>
<!--map-->

To change coordinates(map is centering with them) replace values in data-lat="40.677763" data-lng="-73.943869".

If you want to change maps zooming - replace value in data-zoom="13".

To change marker coordinates replace values in data-lat="40.675653" data-lng="-73.936754" in a element with class="marker".

You can change text, that is shown after clicking on marker.
Just replace text in data-string="1. Here is some address or email or phone or something else...".

Be careful with that changes! Incorect values could cause serious errors.

<div class="copyright">

Page configurations

If you want your page to be loaded with another color (not by picking color after page loading) find block: 

<!--Theme config pop-up-->
<div class="theme-config">
    <!--color picking-->
    <div class="title">Select Color:</div>
    <div class="colors-wrapper">
        <input class="jscolor" onchange="updateScheme(this.jscolor)" value="FDA1A4"/>
    </div>
    ...

Replace value="FDA1A4" with value of your color in HEX format.

If you want your page to be loaded with another fonts(not by picking fonts after page loading) find same block.
Here you'll see the following code: 

...
<!--fonts picking-->
    <div class="simple-select">
        <select id="font-title">
            <option value="Anton@font-1-1">Anton</option>
            <option selected value="Josefin+Sans:300,400@font-1-2">Josefin Sans</option>
            <option value="Hind:700@font-1-3">Hind</option>
            <option value="Arvo:700@font-1-4">Arvo</option>
        </select>
    </div>
    <div class="empty-space marg-lg-b10"></div>
    <div class="simple-text">
        <b>Text</b>
    </div>
    <div class="empty-space marg-lg-b10"></div>
    <div class="simple-select">
        <select id="font-text">
            <option value="Kreon:400,700@font-2-1">Kreon</option>
            <option selected value="Martel:400,700@font-2-2">Martel</option>
            <option value="Maven+Pro:400,700@font-2-3">Maven Pro</option>
            <option value="Cabin:400,700@font-2-4">Cabin</option>
        </select>
    </div>
...

For titles font in block with id="font-title" youl'll see option with selected attr.
Remove it and add to one of the options.

For regular text font do the same, but in block with id="font-text".
